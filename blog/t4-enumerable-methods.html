Don't forget the map!

  • Peter Jahnes
  • 10/12/2014
  • Map Method in ruby

The map method is used on an array or hash. It will do something, almost anything really, to each element in that array or hash. It can be very useful when you need to modify each item in the array and at the same time preserve the original array or hash. The modifier can be a simple function or it can be a complex set of instructions. The syntax is as follows.

$ array.map{ |value| value instructions}

The array is being maped for each value. It then applys the instructions to each value. Here is a more syntacticly correct and useful example.

$ hash.map{ |key,value| value + 1 }

The only difference is that you have a key as well as the value. The key can also receive instructions or you can even manipulate both together in a more complex function.

If you think about it from another prospective, namely mine, It looks, smells, and acts like a loop. It is just a loop that iterates over a group of elements. The map method can replicated for the most part using an each loop. This is very easily shown my the more complex syntax using 'do' and 'end'.

$ Array.map do |value|
$ value + ' hello'
$ end

I hope this has been a help!
